# Telegram Echo Bot

Учебный Telegram-бот, который повторяет сообщения пользователя.

## Возможности

- `/start` — приветствие
- `/ping` — проверка, что бот работает
- любое текстовое сообщение — бот отправляет этот же текст обратно
